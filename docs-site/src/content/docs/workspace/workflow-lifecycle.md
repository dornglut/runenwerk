# Workflow Lifecycle

This document is superseded by [Engineering Workflow](engineering-workflow.md).

Historical links remain valid, but this path does not define active process.

Operational work now uses:

```text
proposed
active
blocked
done
deferred
```

Durable documents may additionally be `accepted`, `superseded`, or `archived`. New states require a distinct review, release, or compatibility decision rather than a process-only distinction.
