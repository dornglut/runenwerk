# Complete Design Gate

This workflow is superseded by [Engineering Workflow](engineering-workflow.md).

Historical reports may continue linking this path. It does not authorize new work.

The durable principle retained from this document is:

```text
shared, public, architectural, and extraction work requires explicit target ownership, dependency direction, alternatives, migration, decomposition, acceptance evidence, and named non-owned responsibilities before implementation
```

The replacement workflow keeps that requirement for the `architectural or extraction` work class while allowing routine and significant changes to use proportionate records.
