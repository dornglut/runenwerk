# Complete Investigation Gate

This workflow is superseded by [Engineering Workflow](engineering-workflow.md).

Historical reports may continue linking this path. It does not authorize new work.

The durable principle retained from this document is:

```text
architecture, extraction, public API, dependency, and ownership decisions must be grounded in current code, tests, accepted authority, alternatives, and explicit uncertainty before implementation
```

The replacement workflow applies that principle through the `architectural or extraction` work class without requiring a universal dossier or fixed matrix set for every change.
