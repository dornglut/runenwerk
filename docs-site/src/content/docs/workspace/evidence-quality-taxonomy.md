# Evidence Quality Taxonomy

This workflow is superseded by [Engineering Workflow](engineering-workflow.md).

Historical reports may continue linking this path. It does not authorize new work.

Use concrete evidence labels when they affect a decision:

```text
connector inspection
manual source inspection
local command result
CI result
user-reported result
not verified
```

A fixed taxonomy, confidence matrix, or truth certificate is not required. Claims must identify the evidence that supports them and any material gap.
