## Resolver Evidence

- `docs-site/src/content/docs/reports/execution-evidence/pt-ui-program-architecture/**`

Evidence records are inputs to truth certification, not proof by themselves. Current resolver-backed records must name concrete subjects, validation command results, and subject digests.

## Run Ledgers

- `docs-site/src/content/docs/reports/track-execution-runs/pt-ui-program-architecture/**`
- `docs-site/src/content/docs/reports/track-execution-runs/pt-test/**`

Successful/current ledgers remain diagnostic history. Failed historical ledgers are retained as diagnostic history unless a later retention policy archives them through a dedicated migration. Tests must not create new `PT-TEST` ledgers in the repository tree; test ledgers belong in temporary roots.

## Product Code And Tests

- `domain/ui/**`
- `Cargo.toml`
- `Cargo.lock`

These are current implementation truth for the UI architecture track. They must be inspected by conformance verifiers directly; generated reports cannot substitute for this code truth.

## Workflow Code And Tests

- `tools/workflow/truth/**`
- `tools/workflow/execution/**`
- `tools/workflow/track_sources/**`
- `tools/workflow/tests/**`
- `tools/workflow/test_workflow.py`
- `tools/workflow/production_goal.py`
- `tools/workflow/production_state.py`
- `tools/workflow/production_track_cli.py`
- `tools/workflow/track_control_cli.py`

Workflow changes are source truth for the execution and verification system. The split workflow test suite is now the safety evidence surface; `tools/workflow/test_workflow.py` remains only a small layout guard.

## Ignored And Local Artifacts

- `.runenwerk/**`
- local caches such as `target/`, `.pytest_cache/`, `__pycache__/`, and editor/system artifacts

Only local/cache artifacts should be ignored. Generated authority, evidence, certificates, ledgers, and reports must stay visible unless an explicit archival workflow moves them.

## Drift Rule

The worktree is acceptable only when every dirty file is in one of the categories above, validation passes, truth certificates are current, and `task truth:audit -- --track PT-UI-PROGRAM-ARCHITECTURE` reports zero findings, zero gaps, zero risks, and zero drift.
