# PM-EDITOR-UX-007 All Registered Visible Surface Wave Closeout

## Summary

`PM-EDITOR-UX-007` / `WR-117` completes a runtime-proven registered editor
surface readiness wave. The Editor UX Story Lab now carries typed
registered-surface evidence for every registered surface definition, and the
app-owned manifest validates product, fallback-only, diagnostic, and
hidden-until-productized readiness evidence.

The runtime proof covers 45 registered-surface story runs and 44 unique
registered surface definitions. The extra story is the richer Material Graph
Canvas product story, which intentionally shares the Material Graph Canvas
surface definition with the generated registry sweep story.

This closeout does not claim game UI readiness or final local-native no-gap
certification. Those remain visible in `PM-EDITOR-UX-008` and
`PM-EDITOR-UX-009`.

## Implementation Summary

- `domain/editor/editor_shell/src/story_lab/surface_wave.rs` module
  `surface_wave` adds `EditorUxRegisteredSurfaceEvidence`,
  `registered_surface_evidence`, readiness-specific artifact/state/route/check
  requirements, and tests covering product, fallback-only, diagnostic, and
  hidden surface categories.
- `domain/editor/editor_shell/src/story_lab/mod.rs` module `story_lab` exports
  the surface-wave evidence contract.
- `domain/editor/editor_shell/src/story_lab/story.rs` struct `EditorUxStory`
  adds optional `registered_surface_evidence` and
  `with_registered_surface_evidence`.
- `domain/editor/editor_shell/src/story_lab/catalog.rs` function
  `surface_story` attaches registered-surface evidence to every generated
  registered surface story; function `material_graph_canvas_story` also attaches
  registered-surface evidence to the richer Material Graph Canvas product story.
- `apps/runenwerk_editor/src/shell/editor_lab_evidence/mod.rs` enum
  `EditorLabEvidenceArtifactKind` adds `SurfaceReadinessReport`.
- `apps/runenwerk_editor/src/shell/editor_ux_story_lab/manifest.rs` struct
  `EditorUxEvidenceRun` adds typed registered-surface evidence, and
  `EditorUxEvidenceManifest::validate` rejects missing surface contracts,
  missing surface reports, identity/readiness mismatches, and incomplete
  artifact/state/route/provider evidence.
- `apps/runenwerk_editor/src/shell/editor_ux_story_lab/runner.rs` function
  `run_story` emits PM007 surface-readiness artifacts and runtime proof.
