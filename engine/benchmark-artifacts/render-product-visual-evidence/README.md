# Render Product Visual Evidence Artifacts

This folder stores raw or reproducible artifact references for WR-078 renderer
product visual evidence. Human-readable reports live under
`docs-site/src/content/docs/reports/benchmarks/render`.

The WR-078 slice records deterministic artifact references for particle/VFX,
world visual, and deformation handoff evidence. Future hardware or visual
capture runs can add raw benchmark output next to this README without making
renderer inspection the source of product animation, VFX, water, weather,
vegetation, field, or fallback truth.
